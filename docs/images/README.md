# docs/images

This directory holds screenshots and diagrams referenced by the documentation pages.

## Required screenshots

The following images are referenced in the documentation and should be added here when available:

| File | Description |
|---|---|
| `control-center-resources.png` | Control Center open on the Resources tab, showing RAM/VRAM meters, Loaded Models, and Queue panels |
| `control-center-onboarding.png` | First-launch onboarding dialog prompting the user to choose a base model |
| `control-center-agents.png` | Agents tab with at least one running agent selected in the list |
| `control-center-manifest.png` | Manifest panel showing approved and pending network-access policies for a selected agent |
| `control-center-models.png` | Models tab showing the model catalog with one model highlighted as active |
| `sandbox-shell-prompt.png` | Terminal after running `bubblehub shell`, showing the sandbox shell prompt |
| `network-access-prompt.png` | Terminal showing the interactive network-access prompt that appears when an agent tries to reach a new host |
| `openclaw-running.png` | OpenClaw running inside the BubbleHub sandbox shell |

## Screenshot guidelines

- Use **PNG** format at **1280 × 800** or similar 16:10 resolution for consistency.
- Crop to show only the relevant area of the screen.
- Do not include personal information, API keys, or host-specific paths in screenshots.
- For retina/HiDPI displays, a `@2x` variant (e.g. `control-center-resources@2x.png`) is welcome but not required.
